ABC Flowchart

Differential Reinforcement Planning Tool — QuickStart Nova Scotia
How to use: Map a common antecedent (trigger) to tiers of child behaviour. 3-tier (default): Red (not workable — discourage), Orange (current skills — moderate reinforcement), Green (target skills — heavy reinforcement). 2-tier: Orange + Green only (when "not workable" tier isn't needed). The gradient shifts child behaviour toward target skills over time. You can create multiple flowcharts, toggle tiers per flowchart, and print any of them.
